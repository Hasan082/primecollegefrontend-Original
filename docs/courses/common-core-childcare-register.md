# Course: Common Core Training for the Childcare Register

**Provider:** Prime College  
**Type:** CPD Accredited Course  
**Estimated Learning Time:** 15–18 hours CPD  
**Target Audience:** Registered childminders, individuals applying to Childcare Register, early years practitioners, childcare assistants, CPD seekers

## Module Structure Pattern

Each module follows a consistent pedagogical pattern:

1. **AI Voice Script** – Welcome narrative explaining the module topic
2. **Learning Outcomes** – Measurable outcomes listed at the start
3. **Sections** – Content divided into numbered sections with:
   - Educational content with professional explanations
   - **Real Childminder Examples** – Practical scenarios illustrating concepts
   - **Reflection Prompts** – Questions requiring written reflection (2-3 sentences)
   - **Knowledge Checks** – Multiple choice questions (A-D) with correct answer marked
   - **Recommended Resources** – External links (gov.uk, NSPCC, Ofsted, PACEY, NHS, etc.)

## Platform Implications

This course structure informs how the platform should handle:

1. **Module delivery**: Sequential modules with structured content (not live teaching)
2. **Knowledge checks**: MCQ format (A-D options) with auto-marking capability
3. **Reflection activities**: Text input areas requiring written responses (trainer-reviewed)
4. **Resource library**: External links + downloadable resources per module
5. **Progress tracking**: Per-module completion (knowledge check + reflection = module done)
6. **Final assessment**: Comprehensive MCQ covering all modules with pass mark
7. **Completion flow**: All modules → Final assessment → Evaluation form → Declaration → Certificate
8. **CPD hours tracking**: 15-18 hours logged against learner record
